# 📚 项目概览

## 🎯 项目目标

创建一个基于DeepSeek的智能arXiv论文自动爬取和分析系统，能够：

- 🔍 根据自定义关键词自动爬取arXiv最新论文
- 🤖 使用DeepSeek AI生成论文摘要和研究洞察
- ⏰ 支持定时自动执行和手动触发
- 💾 本地数据库存储，支持去重和搜索
- 📊 自动分析研究趋势和热门主题
- 🔄 支持论文比较和深度分析

## 📁 核心模块

### 1. 配置管理 (`config.py`)
- 环境变量加载
- DeepSeek API配置
- 爬取参数设置
- 日志配置

### 2. 数据库管理 (`database.py`)
- SQLite数据库操作
- 论文数据模型
- 去重机制
- 查询和搜索

### 3. arXiv爬虫 (`arxiv_scraper.py`)
- arXiv API集成
- 关键词搜索
- 数据解析和清洗
- 热门主题提取

### 4. DeepSeek分析器 (`deepseek_analyzer.py`)
- 论文摘要生成
- 研究洞察分析
- 论文比较分析
- 趋势报告生成

### 5. 任务调度 (`scheduler.py`)
- 定时任务管理
- 每日/周度任务
- 状态监控
- 自动化报告

### 6. 用户界面
- **CLI界面** (`cli.py`, `main.py`): 命令行交互
- **Web界面** (`web_app.py`): Flask Web应用
- **API接口**: RESTful API

## 🔄 工作流程

1. **初始化**
   - 加载配置和环境变量
   - 初始化数据库连接
   - 设置日志系统

2. **爬取阶段**
   - 根据关键词构建搜索查询
   - 调用arXiv API获取论文列表
   - 解析论文元数据
   - 存储到数据库（去重）

3. **分析阶段**
   - 调用DeepSeek API生成摘要
   - 分析研究趋势和热点
   - 生成洞察报告
   - 更新论文摘要

4. **输出阶段**
   - 保存每日洞察文件
   - 生成周度报告
   - 更新热门主题列表
   - 发送通知（可选）

## 🛠️ 技术栈

- **后端**: Python 3.8+
- **数据库**: SQLite
- **AI服务**: DeepSeek API
- **Web框架**: Flask
- **任务调度**: Schedule
- **数据处理**: BeautifulSoup4, feedparser
- **配置管理**: Pydantic, python-dotenv

## 🚀 部署选项

### 1. 本地部署
```bash
python setup.py
python start.py --mode cli  # CLI模式
python start.py --mode web  # Web模式
```

### 2. Docker部署
```bash
docker-compose up -d
```

### 3. 服务器部署
- 使用systemd服务管理
- 配置cron定时任务
- 日志轮转配置
